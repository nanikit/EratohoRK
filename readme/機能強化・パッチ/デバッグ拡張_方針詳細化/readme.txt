□사용법

□용어의 설명
판정치  AI가 실제로 판정에 사용한 수치입니다. RAND에서의 결과가 가미되고 있습니다
고정치  판정 시에 고정치로 가감(상태)되어서, RAND가 관련되지 않는 수치입니다
변동치  RAND:X가 가감(상태)되고 있는 것입니다. 이 수치가 AI의 랜덤 놀리라고 됩니다
　　　　변동치는（X-1)으로, 예를 들면 RAND:3이라면, 2가 되어, 고정치보다 최대로 2높아질 가능성이 있다고 하는 일이 됩니다
기대치  고정치＋변동치/2입니다

	※실제의 변동치는 복수의 값의 합계치가 되므로 기대치에 가까워지기 쉬울 것입니다
	또, 플러스와 마이너스를 그대로 더하고 있으므로 수치가 작아도 치우침폭이 큰 경우도 있습니다
	（변동 마이너스와 변동 플러스를 나누고 풀면 좋았지만 아무튼 귀찮게 되었으므로 기력이 솟아 오르면…）
