[Emuera에 대해]
Emuera（에뮤라）는 eramaker의 에뮬레이터＋α입니다.
eramaker.exe와 같은 폴더에 넣어 기동하는 것으로 eramaker.exe 대신에 ERB 스크립트를 실행합니다.
마우스로 조작하거나 간단한 매크로 펑션을 사용하는 것 따위가 생깁니다.


[동작환경]
. NET Framework 2.0 환경이 필요합니다.

「어플리케이션을 올바르게 초기화할 수 없었습니다」라고 하는 에러가 나와 기동 할 수 없는 경우,. NET Framework 2.0이 올바르고 인스톨 되고 있지 않습니다.
Microsoft 다운로드 센터로부터. NET Framework 2.0 런타임을 다운로드해 인스톨 해 주세요.


[사용 방법]
EmueraXXXX.exe를 eramaker.exe와 같은 폴더에 넣어 기동해 주세요.
(XXXX에는 버젼을 나타내는 숫자가 들어갑니다. 또, 환경에 따라서는. exe가 표시되지 않는 것이 있습니다)
기동하지 않는 경우,[동작환경]시에를 읽어 주세요.
기동 직후에 emuera.config, 에러 종료했을 경우에는 emuera.log를 생성합니다.
자세한 것은
http://osdn.jp/projects/emuera/wiki/FrontPage
를 참조해 주세요.
