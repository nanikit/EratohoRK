●好感度による口上変化について
某所でのアンケートでは「好感度上がってから甘々に変化する口上」が希望一位でしたが、
Revの仕様上、好感度により行動自体に偏りが出てくるわけだから、わざわざ書き分けなくてもいいんだ！
と開き直ったため好感度による分岐はしません。まあ、そのうち書き足すかも。

しかし、最初からラブラブしちゃうのもフランちゃんの魅力ですよね。
逆に誰とでもラブラブしちゃうロリビッチ臭が漂ってくれば更に良し。

●これはスゴい！ eratohoRevの革命【浮気破滅システム】企画書

「Rev用 青娥娘々 口上」
お姉さん、人妻、出産経験有り？、悪女、等々とんでもないエロ要素を引っさげて登場した青娥にゃん。
青娥にゃんに身も心も篭絡されたい、青娥にゃんに全財産貢いで借金取りに追われる日々を送りたい、
そんな心の叫びは多々あれども、魅力的な東方キャラの中で一番の嫁だと言う人は少ないはず。

そんな中、この口上ではあなたの一番の嫁の名前を入力するのです（【幽々子】とか【霊夢】とか）
すると度々、その名前を引き合いに出され、

「幽々子様という方がいながら、こんなにおちんちん硬くして、いけない人ですね♪」

と、耳元で囁かれながらシコシコされたり

「ふふっ、どうですか女の蜜壷の味は♪  霊夢様とも交わった事無いのに、
あなた様の陰茎はもう娘々でしかイけなくなってるかもしれませんね（はーと×３）」

とか言われながら青娥にゃんに中出しを強要されるのです。

そこに「イっちゃだめだ、流されちゃだめだ。俺には○○という嫁がいるのに…あ、あぁぁ、ひゅごいのほぉぉぉぉおおお！」
という、プレイヤーとゲーム中の"あなた"にかつて無い程の一体感が生まれるのです。
（Revにはプレイヤーによるこの感覚が少なかった）
今回の口上のあらすじ［2］を書いてる時に思いついた適当な構想ですけど。
