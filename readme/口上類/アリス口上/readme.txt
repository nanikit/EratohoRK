◎처음에
**********
이 구상은 eratohoЯeverse 1.2. 12이후의 추가 요소에 대응한 것입니다
그 이전의 본체 버젼에서는 확실히 불편이 일어납니다

◎게임 모드에 대해
**********************
*NORMAL
캐릭터 번호 0 당신(이름의 변경은 가능)으로서 앨리스에게 조교되는 일이 됩니다
게임 스타트 후 30일을 경과한 시점에서, 일정한 조건을 채우면 엔딩이 됩니다.
맞이하는 엔딩에 따라서는 그대로 종료, 혹은 데이터 로드의 선택이 되어
로드하지 않는 경우는 그대로 종료가 됩니다.
자세한 것은 ending 파일내를 확인해 주세요.

플레이어의 성별 마다의 제한등은 없습니다. 어느 성별에서도 충분히 기능한다고 생각합니다.
또[작은체형 체형]의 경우는 기본적으로 「어린아이의 연령의 정도도 가지 않은 아이」라고 하는 취급이 되어, 개소에 따라서는 분기 합니다.
※[작은체형 체형]은 성교계에의 마이너스 보정이 비교적 강하게 하고였거나
조교 대상이[작은체형 체형]의 경우는 비교적 소프트, 여자 아이의 경우는 아무튼 보통, 남자라면 약간 뻑뻑한 표현이 많을지도 알지 못하고

독심술이 있는 경우는 마음의 소리를 들을 수 있을 때가 있습니다

오프닝을 경유하지 않으면 엔딩에는 이르지 않는 사양입니다.
또, 어느 정도 이상의 낡은 데이터로 중도부터 개시했을 경우는, ED에 이르지 않을 가능성이 있습니다.
※（데이터의 호환성은 가까운 시일내에 없어지는 것이라고 생각됩니다）

*SANDBOX
기본적으로 NORMAL에 준한 내용. 도구 자동 구입등의 기능이 실장되고 있습니다.
OP는 범용 OP를 불러, 그 외 일부 이벤트의 호출은 죽여 있습니다.
이 모드의 경우 엔딩은 없습니다.

*EXTRA
조교 대상(즉 주인공)에 사쿠야, 조교자（주인의 경우 한정）에 앨리스를 선택했을 경우에게만, 전용의 구상이 불려 갑니다.
대상 한정 구상은 파일을 함수 나누고 하고 있어, 완전하게 독립해 기능합니다.
ver 1.53으로 엔딩을 추가. 도중의 이벤트의 선택지에 의해 ED까지의 루트가 분기 합니다.

자세한 것은 extra 파일내를 확인해 주세요.

조교 대상으로 사쿠야 이외를 선택했을 경우는 OP는 범용 OP를 불러, 도중의 정합성을 해친다고 생각된다
몇개의 이벤트 호출은 죽여 있습니다. 또, 이 경우는 엔딩은 없습니다.

어느 게임 모드도 신규 스타트 추천이 되고 있습니다.

일부의 정합성을 해친다고 생각되는 커맨드나 문제 있음이라고 생각되는 커맨드, 개인적으로 용인 할 수 없는 커맨드는
구상측에서 죽여 있습니다. 양해해 주십시오.


◎버젼 개요
****************
·바리안트 본체 1.2. 12에 추가된 오늘의 방침 "귀여워해"에 대응
·EV16, 대만족 보너스에 분기의 추가와 가필
·한정 없음 구상과 소개미 구상으로 대응
·애무계 ACT(가슴 애무/입으로 애무/키스/파이즈리 한다)에 가필
·절정 구상(페라로 사정)의 개소에 가필

만족 보너스[11]귀여워해서는 조교자의 호감도 2000, 조교 대상의 호감도 1000으로부터 선택 가능
애무계의 가슴 애무, 입으로 애무, 파이즈리등에+보정됩니다
지금까지 발생하지 않았던 이것들 ACT도 만족 보너스[11]를 선택하는 것에 의해 많이 발생합니다
소질:혀사용의 취득이나 풍유화로 한층 더+보정
페라로 사정때, 구음경험 50＆조교자 상태 변화 "정욕"시에는 딥스로트에 파생
지금의 사양으로 가장 많은 구상을 볼 수 있는 자캐릭터는 아마[작은체형][후타나리]가져 근처

※그 외 갱신된 내용의 자세한 것은 version.txt 파일을 확인해 주세요.
