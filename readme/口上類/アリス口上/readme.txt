◎はじめに
**********
この口上はeratohoЯeverse 1.2.12以降の追加要素に対応したものです
それ以前の本体バージョンでは確実に不具合が起きます

◎ゲームモードについて
**********************
*NORMAL
キャラ番号0 あなた(名前の変更は可能)としてアリスに調教される事になります
ゲームスタート後30日を経過した時点で、一定の条件を満たせばエンディングとなります。
迎えるエンディングによってはそのまま終了、もしくはデータロードの選択となり
ロードしない場合はそのまま終了となります。
詳細はendingファイル内を確認して下さい。

プレイヤーの性別毎の制限等はありません。どの性別でも十分に機能すると思います。
また[小柄体型]の場合は基本的に「年端もいかない子供」という扱いになり、箇所によっては分岐します。
※[小柄体型]は性交系へのマイナス補正が割りと強めだったり
調教対象が[小柄体型]の場合は割りとソフト、女の子の場合はまぁ普通、オトコだとややきつめの言い回しが多いかも知れず

読心能力がある場合は心の声が聞ける時があります

オープニングを経由しないとエンディングには至らない仕様です。
また、ある程度以上の古いデータで中途から開始した場合は、EDに至らない可能性があります。
※（データの互換性は近いうちに失われるものと思われます）

*SANDBOX
基本的にNORMALに準じた内容。道具自動購入等の機能が実装されています。
OPは汎用OPを呼び、その他一部イベントの呼び出しは殺してあります。
このモードの場合エンディングはありません。

*EXTRA
調教対象(つまり主人公)に咲夜、調教者（主人の場合限定）にアリスを選んだ場合にのみ、専用の口上が呼び出されます。
対象限定口上はファイルを関数分けしており、完全に独立して機能します。
ver 1.53でエンディングを追加。途中のイベントの選択肢によりEDまでのルートが分岐します。

詳細はextraファイル内を確認して下さい。

調教対象に咲夜以外を選択した場合はOPは汎用OPを呼び、途中の整合性を損なうと思われる
幾つかのイベント呼び出しは殺してあります。また、この場合はエンディングはありません。

いずれのゲームモードも新規スタート推奨となっています。

一部の整合性を損なうと思われるコマンドや問題アリと思われるコマンド、個人的に容認出来ないコマンドは
口上側で殺してあります。ご了承下さい。


◎バージョン概要
****************
・バリアント本体1.2.12に追加された今日の方針"かわいがって"に対応
・EV16,大満足ボーナスに分岐の追加と加筆
・限定無し口上と咲アリ口上で対応
・愛撫系ACT(胸愛撫/口で愛撫/キス/パイズリする)に加筆
・絶頂口上(フェラで射精)の箇所に加筆

満足ボーナス[11]かわいがっては調教者の好感度2000,調教対象の好感度1000から選択可能
愛撫系の胸愛撫、口で愛撫、パイズリ等に+補正されます
今まで発生しなかったこれらACTも満足ボーナス[11]を選択することによってモリモリ発生します
素質：舌使いの取得や豊乳化でさらに+補正
フェラで射精の際、口淫経験50＆調教者の状態変化"情欲"の時はディープスロートに派生
今の仕様で最も多くの口上を見られる自キャラは恐らく[小柄][ふたなり]持ちあたり

※その他更新された内容の詳細はversion.txtファイルを確認して下さい。
