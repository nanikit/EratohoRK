■시각 애니메이션 효과와 당구상에 대해 간단한 해설

[1] 캐릭터 등장 컷 인
  *OP, ED로 사용
  *조수 소환, 조교자 교대시에 본체측에서 사용
  *off로 간이 표시

[2] BGM 컷 인
  *이벤트로 사용
  *off로 간이 표시

[3] PLAM 획득시의 애니메이션
  *본체측만으로 사용

[4] 탈의·착의시의 애니메이션
  *본체측만으로 사용

[5] 낮·밤에서의 키 경치 변화
  *본체측만으로 사용
  (대상 한정 구상 일부 개소에서 BGCOLOR에서의 변경 있어)

[6] 전화면 컷 인
  *OP 및 이벤트 관계로 사용
  *off로 간이 표시되고 WAIT가 걸린다

[7] 사정 때 등의 화면 플래시 효과
  *대상 한정 구상, 절정 구상 부분에서 사용
  *한정 없음 구상 SOURCE_1(조교 대상 사정 부분)으로 본체측이 사용
  *off로 비표시
  (SOURCE_5, 조교자 사정 부분에서는 본체측의 처리 없음)

[8] 화면 진동 효과  
  *COM20허리를 흔드는 및 절정 구상으로 사용  
  *대상 한정 구상 ACT로 일부 사용
  *off로 비표시

[9] 문자의 슬라이드 애니메이션 효과
  *한정 없음 구상 ED로 사용
  *off로 간이 표시

[10] 문자의 페이드 인·아웃 효과
  *주로 이벤트 관계로 사용
  *대상 한정 구상에서는 조교중에도 사용
  *off로 간이 표시

동작 확인은[6]의 전화면 컷 인만 OFF로 하고 있습니다
기본적으로 상기의 설정을 전제로 해서 만들어지고 있으므로, 설정 하기에 따라 작자의 의도와 다른 것이 될 수 있습니다
특히 FLASH는 사정 관계로 비교적 이치가 맞지 않게 되는 염려있어
구상의 전화면 컷 인계는 모두 강제 표시가 되어 있습니다
CALLF ANIME_CONFIG("TIMES", 1)로 강제 표시하고 있으므로,
아무래도 받아들일 수 없는 경우는 자력으로 삭제하든지 comment out 해 주세요
