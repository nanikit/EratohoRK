◎EXTRAモード用口上について

EXTRAモードで調教対象を咲夜、調教者をアリスとした場合のみ専用の口上が呼ばれます。
それ以外の場合は一般口上が呼び出されますが、整合性の観点から専用OPを含む一部イベントは殺してあります。
またその場合、エンディングはありません。
現状の仕様だとSANDBOX用として解放してもいい気もしますがとりあえずはEXモードのみって事で。

バリアント本体のEXTRAモードの説明ではストーリーが無いとなっていますが、それは汎用OP等の事で
各種イベントは条件に適えば普通に表示されるようになっています。

人によっては一部アニメーション関係の演出がくどいと感じられるかも知れません。
その場合は本体のコンフィグで対応して下さい。
予め同梱のanime_configファイル内を確認してもらうと良いかも知れません。
口上側の全画面カットインはコンフィグに関わらず強制表示となっています。ご了承下さい。

一部の整合性を損なうと思われるコマンドは口上側で封印してあります。
また、日常イベントと奴隷の妊娠(現状機能してるか不明ですが)はコンフィグの設定に関わらず
封印する処置をしてあります。何度設定してもoffになるのは口上側の仕様なので本体側に文句は言わないで下さい。
また、助手召喚は口上側で殺してあります。

各場面の描写はかなり濃い目になっており、テンポは非常に悪いと思われます。
今までのNORMALモード同様のものを期待している場合、それは裏切られます。
サクサク遊びたいという人にはEXTRAモードはお勧めしません。
まぁ本編のおまけとして割り切って楽しめる人向け。
といいつつ、一般口上部分より遥かに気合が入ってるのは秘密。

※ver 1.61でEDが50日以降に発生するように改変
※Ver 1.61からEndingによってはED後の継続が可能なようになりました

現在GOODとBADの二種のみ。
途中の選択肢によりルートが分岐し各種イベントはそのルート専用のものとなります。
50日経過した上で所定のイベントを回収しているとエンディングになります。
エンディングによってはデータロード、もしくはそのまま終了となります。予めご了承下さい。

また調教中止時のイベントは現在工事中なので一部のみ解放。その他は封印してあります。
ご了承下さい。

■エンディングについて（ネタバレを含みます）

相思相愛時初回イベントの選択肢によってEDまで枝分かれします。
選択肢[0]でルートＢ
選択肢[1]でルートＡ
となり、以降イベントはルート毎に別個に処理されます。
ルートＡ、ルートＢ等の名称は管理の都合でこちらが便宜上用いているだけなので
ゲーム中に目にする事はありません。

現状エンディングは2つ。最終的にはもう一つのED(TRUE END)の追加を予定

1, (Ending No.6　GOOD END)
	・選択肢[1]を選び、ルートＡに入った場合このEDになります。
	・所定のイベントを回収して40日目を過ぎればEDが呼び出されます。
	・ED中に選択肢により内部で分岐しますが最終的には同じものになります。
	・とりあえず、収まるところに収まったGOOD END。
	・エンディング後のプレイ継続は可能です。
	
2, (Ending No.7　BAD END)
	・選択肢[0]を選び、ルートＢに入った場合このEDになります。
	・所定のイベントを回収して40日目を過ぎればEDが呼び出されます。
	・選択肢のない一本道。悲劇的な結末なので注意が必要かも知れず。
	・とはいえ、後味の悪いものではないという自負はあります。
	・このエンディングの場合ゲーム終了、またはデータロードとなりプレイ継続は出来ません。
	・悪しからず。

尚エンディングの番号は一般口上からの通し番号となっています。
TRUE ENDは全ての箇所の手入れが終わり、完成と言える時点で追加する予定。
相思相愛時のご褒美による調教中止イベントでTRUE ENDに向けて色々と匂わせていますが、最終的な形は変わるかも知れず。
基本的に咲夜さんの素質は調教対象としてはかなりキツいんで、気長にやるのがいいと思います。
なかなか思うようにならないのもRevの面白さの一つだと思うんで。
