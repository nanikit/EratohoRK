;◎口上内容
;----------------------------
;『せつめい』
;	リリーちゃんに甘々でえっちなイタズラをされちゃう口上です。
;	実際素質がいい子過ぎて、プレイヤーがコントロールしないとなかなか酷い事してくれない気がします……
;	『魔力＝精』という表現は大妖精口上様から、同じ妖精なので一部認識に共通点を持たせてあります。
;
;『小ネタ』
;	・着衣状態、妊娠経験、男女別で適度に派生
;	・一部大ちゃん口上実装
;	・最初に選ぶご主人様/途中参加のご主人様で分岐
;
; ＊個人メモ
;　 死に口上回避の為イベントで素質付与
;　 調教済み：調教者が噴乳やアナル絶頂できるようになる、かも（「EV13_調教終了_K15.ERB」386行目～）
