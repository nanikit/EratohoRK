◎はじめに
**********
この口上はeratohoЯeverse1.1.16用です
1.1.16以前のバージョンでは不具合が起きます

◎ゲームモードについて
**********************
*NORMAL
キャラ番号0 あなた(名前の変更は可能)として雛に調教される事になります。
調教対象を『オトコ』とした場合のみに口上が呼び出される仕様。おんな、ふたなりでは口上は表示されません。
オトコ以外の場合、開始時の口上選択にも表示されません。
ゲームスタート後30日を経過した時点で、一定の条件を満たせばエンディングとなります。
迎えるエンディングによってはそのまま終了、もしくはデータロードの選択となり
ロードしない場合はそのまま終了となります。
詳細はendingファイル内を確認して下さい。

仕様上、オープニングを経由しないとエンディングには至りません。
既存のデータで中途から開始した場合は条件を満たしてもEDに至らない可能性があります。

*SANDBOX
基本的にNORMALに準じた内容。道具自動購入等の機能が実装されています。
OPは汎用OPを呼び、その他一部イベントの呼び出しは殺してあります。
このモードの場合エンディングはありません。

*EXTRA
OPは汎用OPを呼び、整合性を損なうと思われる幾つかのイベント呼び出しは殺してあります。
またこの場合はエンディングはありません。

調教対象をオトコ以外にした場合全てのモードで口上は表示されないので、ご了承下さい
また性交系(調教者→調教対象)は封印してありますので、ふたなり化やペニスバンドは無駄になります。

いずれのゲームモードも新規スタート推奨となっています。


◎あそびかた
************
ただ普通にプレイしていると"手で愛撫"ばかりになりがちです
そういう場合はSHOPで素質を購入しましょう
[舌使い]はフェラやキスを誘発します
[腰使い]は性奉系や素股を誘発します
[巨乳]はパイズリを誘発します

その他の素質、技能、道具の有無でも動作が変わるので試してみると良いかと
あと服は基本的に各種行為の妨害要素になるので、面倒なのが嫌な人は(二人共)全裸で過ごすのが良いです
