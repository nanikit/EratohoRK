■시각 애니메이션 효과와 당구상에 대해 간단한 해설

[1] 캐릭터 등장 컷 인
*OP로 사용  
*조수 소환, 조교자 교대시에 본체측에서 사용
*off로 간이 표시

[2] BGM 컷 인
*이벤트로 사용
*off로 간이 표시

[3] PLAM 획득시의 애니메이션
*본체측만으로 사용

[4] 탈의·착의시의 애니메이션
*본체측만으로 사용

[5] 낮·밤에서의 키 경치 변화
*본체측만으로 사용

[6] 전화면 컷 인
*본체측만으로 사용
*off로 간이 표시

[7] 사정 때 등의 화면 플래시 효과
*절정 구상 부분에서 사용
*off로 비표시

[8] 화면 진동 효과  
*COM20허리를 흔드는 및 절정 구상으로 사용  
*ACT로 일부 사용
*off로 비표시

[9] 문자의 슬라이드 애니메이션 효과
*엔딩으로 사용
*off로 간이 표시

[10] 문자의 페이드 인·아웃 효과
*주로 이벤트, 조교 관계로 사용
*off로 간이 표시

동작 확인은[6]의 전화면 컷 인만 OFF로 하고 있습니다
기본적으로 모든 효과가 ON인 일을 전제로 제작하고 있습니다
