■視覚アニメ効果と当口上について簡単な解説

[1] キャラ登場カットイン 
*OPで使用　
*助手召喚、調教者交代時に本体側で使用
*offで簡易表示

[2] ＢＧＭカットイン 
*イベントで使用
*offで簡易表示

[3] PLAM獲得時のアニメーション 
*本体側のみで使用

[4] 脱衣・着衣時のアニメーション 
*本体側のみで使用

[5] 昼・夜での背景色変化 
*本体側のみで使用

[6] 全画面カットイン 
*本体側のみで使用
*offで簡易表示

[7] 射精時等の画面フラッシュ効果 
*絶頂口上部分で使用
*offで非表示

[8] 画面振動効果　
*COM20 腰を振る及び絶頂口上で使用　
*ACTで一部使用
*offで非表示

[9] 文字のスライドアニメ効果 
*エンディングで使用
*offで簡易表示

[10] 文字のフェードイン・アウト効果
*主にイベント、調教関係で使用
*offで簡易表示

動作確認は[6]の全画面カットインのみOFFでしています
基本的に全てのエフェクトがONである事を前提に製作しています

■口上推奨コンフィグ
現在の仕様では開始時に口上推奨のコンフィグ設定が可能ですが、その内容を以下に示しておきます

[一般設定]
＊主人公の読心能力
・心が読めない

＊素質の表示分類
・元の設定を変更しない

＊日常イベント
・オン

＊調教者の妊娠判定
・OFF

＊調教対象の妊娠判定
・OFF（調教対象がオトコの場合専用なので）

＊AI自由度（SANDBOX用）(0=余計なことしない、1=人員追加、2=技巧上昇、4=技能取得、8=道具錬成、15=全てを自動で行う)
・人員増加、技能取得、道具練成を有効に

[アニメ設定]
＊キャラ登場カットイン
・ON

＊ＢＧＭカットイン
・ON

＊PALAM獲得アニメ
・ON

＊脱衣アニメ
・ON

＊昼夜の背景色変化
・ON

＊全画面カットイン
・OFF（口上側の処理は強制表示される）

＊フラッシュ効果
・ON

＊振動効果
・ON

＊文字のスライド
・ON

＊文字のフェードイン・アウト
・ON
