eratohoRev0.60用、犬走椛口上です。
現段階では、調教者は「あなた（オトコ）」のみを想定したところまでしか完成していません。（極々一部のみ女にも対応）
椛が好きだから書いた。反省はしていない。
しかし、テンプレがなければ作ろうと決心することもなかったかもしれません。
テンプレートを作ってくださったぱの人には本当に心から感謝申し上げます。
なお、この口上は0.60時点での使用を前提としているため、それ以降の版では不具合が生じる可能性もありますがご了承ください。


当口上の椛の特徴
・口調はどことなく犬っぽく、時々狼
・いじけて「の」の字を指で書く癖がある。もちろん元ネタは原作の弾幕
・同じ犬系キャラである「能美クドリャフカ（リトルバスターズ！）」の口調が混じっているかも（「わふー」など）
・下着はサラシとフンドシ。別に男装しているわけではなく、あの服にブラやドロワは似合わないという作者の主観から。
・二次創作でよく見かける設定である「文とは知り合いで先輩後輩の関係」となっている（原作では一切描写なし）

改編・転載についてはご自由に行ってくださって結構です。
椛は原作でも台詞がなく、犬走椛というキャラの受け取り方は人それぞれでしょう。
「椛はもっとMだ！」「いやSだ！」「もっと淫乱だ！」「むしろボクっ娘だ！」
など、色々な妄想ができるため、私の口上に違和感を持つ方もいるでしょう。
自分だけの椛を書きあげて思う存分もみもみするのもまた貴方のジャスティスです。

ぱの人へ。口上まとめへの編入は一向に構いません。むしろよろしくお願いします。


使い方
EVENT_K41をそのままERBの口上フォルダに入れてご使用ください。


同梱パッチについて
口上では、マイジャスティスに従い椛の下着をふんどし＆サラシ前提で書いてしまったため、
初期表示の「ドロワーズ」「ブラジャー」と矛盾するということに今になって気が付きました。
そのため、椛の服装初期設定をドロワーズ→褌、ブラジャー→さらしに変えるためのパッチです。
使い方は、元のファイルと置き換える形でそれぞれのフォルダにぶちこむだけです。
なお、パッチを使わずとも、口上において褌、サラシとあるのにゲーム上はブラやドロワ表記という
矛盾が生じるだけで、ゲーム内容に変更はありません。
下手に元のファイルを置き換えるのが怖い、という方はパッチを使わなくても特に支障はないかと思われます。
（口上が出てきたときに脳内で補完していただければ）
ただし、このパッチを入れても、コンティニューでは表記は変わりませんので、表記を変更したい場合はニューゲームを選択し、最初からキャラの衣装データを読み込ませる必要があります。ご注意ください。


今後の予定
元々の口上テンプレートではフェラやパイズリ、素股などの奉仕系コマンドに対する個別口上がないため、
独自に追加していきたいと思います。やはり献身的な椛には奉仕口上を充実させたい。
あとはJ用ナズーリン口上にも手を出し始めたので、しばらくは並行作業となりますが、
社会人ということもありますので年末にかけて忙しくなり、作業は遅くなっていくかもしれません。
