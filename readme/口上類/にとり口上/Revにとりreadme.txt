eratohoRev 전용에 있어 구상 ver0. 2 readme

·개요
  eratohoRev 전용의에 있어 구상 파일입니다.
  나의 안의 것에 새의 이미지를 마음껏 표현해 보았습니다.
  「에 새는 그런 일 말하지 않는다」 「누구 너」…… 미안합니다.
  개변, 가필등은 그 취지 명기해 받을 수 있으면, 자유롭게 아무쪼록.
  eratohoRev, Rev 구상의 작자의 여러분, 다양하게 참고에 하도록 해 받았습니다. 정말로 감사합니다.

·사용법
  eratohoRev의 구상 폴더에 ERB 파일을 돌진해 주세요.

·주의
  에러등 나올지도 모릅니다. 그 때는 보고를 부탁합니다.
  문장의 연결이 이상한 곳이 있을지도 모릅니다만, 웃어 간과해 주세요.
  기분 전환에 만들기 시작한 것으로, 그다지 기대하지 않고 즐겨 받을 수 있으면 다행입니다.

·향후의 예정
  공백을 묻는다.
  현시점에서 아라이먼트 정부의 분기 뿐인 것으로, 서로 사모함 등분기의 추가.
  패턴 증가.
