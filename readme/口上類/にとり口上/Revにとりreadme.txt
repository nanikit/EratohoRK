eratohoRev専用にとり口上ver0.2 readme

・概要
　eratohoRev専用のにとり口上ファイルです。
　私の中のにとりのイメージを思うがままに表現してみました。
　「にとりはそんなこと言わない」「誰てめぇ」……すみません。
　改変、加筆等はその旨明記していただければ、ご自由にどうぞ。
　eratohoRev、Rev口上の作者の方々、色々と参考にさせていただきました。本当にありがとうございました。

・使用法
　eratohoRevの口上フォルダにERBファイルを突っ込んでください。

・注意
　エラー等出るかもしれません。その際は報告をお願いします。
　文章の繋がりがおかしい所があるかもしれませんが、笑って見過ごしてあげてください。
　気分転換に作り始めたものなので、あまり期待しないで楽しんでいただければ幸いです。
