eratohoReverse# ぽ用天子口上正式版 Ver0.0

・本口上はeratohoReverse# ぽ用の天子口上です。
　前回の更新から早3か月、残り2割が限りなく遠い事を痛感した臨時更新です。
　口上テンプレを射命丸口上からぬえ口上に変更、それに伴いほぼ全ての部分を加筆・修正しました。
　また、砂糖分を従来の500%増しにしました。まさに一人eratohoReverseJ。
　最近の「一旦二次創作から離れ、原作を元にキャラクターを再評価する」流れに則り、安易な二次設定（主にドM）を一旦取り払い、
　彼女の原作での行動・言動から垣間見える、彼女本来の良さを意識した上に個人的な趣味を加える事で本口上は作られております。
　何だかんだ言って心優しく思いやりのあるてんこを暖かい（41℃くらいの）目で見てやってくだされば幸せです。
　
　なお本口上は、今は亡きeratohoReverse# pa用射命丸口上を口上テンプレ代わりにして改変する形で作った物を更にぬえ口上に移植する形で作っています。
　テンプレの更新によりバグは無くなったと思いますが、作り始めの時期の関係で、本家であるぬえ口上の更新部分を人力で同期しています。
　故に同期ミスがあるかも知れません。　…一応全部確認したので大丈夫だとは思いますが。

　また、このバージョンは前回の更新から間が開き過ぎているのに一向に完成が見えない事を考えた上での臨時公開版です。
　まだ埋まっていない（ぬえ口上のセリフが残っている）部分を全て削除して公開してあります。
　十分遊べるくらいに移植・加筆はした上でバックアップは取ってあるので、気にせずにてんこといちゃついて下さい。
