・さらに強靭になった体イベント

 「調教中止時に食べる天界の桃料理に副作用が発生」するもので、料理を食べる度に1/5の確率で発生。
　体が更に強靭になり、体力・気力・精力の最大値が100上昇します。
　さらにこのイベントを計10回発生させてしまうと肉体が天人と同レベルになり、ボーナスとして体力・気力・精力の最大値がさらに1000上昇します。
　さらに強制的に「痛みに強い」と「精力絶倫」を取得してしまいます。
　通称、アミバイベント。
　CVAR:1をアミバイベントの発生回数カウントに使っています。

・食い意地の張りすぎは命に関わるぞイベント

　調教対象がおもらし癖を持っていない事が発生条件です。
　調教中止の際に、1/10の確率で天子が食後のデザートとして妙な飴玉を拾ってきます。
　これを食べると「おもらし癖」を取得してしまいます。
　YMにも存在しない不思議な飴玉ですが、果たしてこれを作ったのは誰なのでしょうか…
　CVAR:2をイベント発生済みフラグに使っています。

・求めよ、さらば与えられんイベント

　天子のふたなり化orふたなり消去時、もしくは巨乳化・貧乳化の際に必要な量の魔力が残っている事が発生条件です。
　魔力を通常の量+1000消費して、調教対象側をふたなり化orふたなり消去、巨乳化・貧乳化させる事ができます。
　CVAR:3をイベント初発生フラグに使っています。
　…ただ、巨乳化・貧乳化の方はまだ未実装です。
　まあ、ふたなりの方をコピペすればすぐ出来るんですが…
